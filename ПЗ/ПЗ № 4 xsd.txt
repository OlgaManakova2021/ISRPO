XML. Практическое задание № 4

Тема: Создание валидных XML документов с помощью XML-схем (xsd).


Указания:

Задания выполнять в среде VS code с установленным плагином «XML Language Support by Red Hat». 
Контроль за корректностью документа осуществлять в браузере.


Задание

1. Создать XML схему (node.xsd), которой будет соответствовать xml-файл из упражнения № 3 (book_error.xml) 
2. Выполнить подключение XML схемы к указанному xml-файлу, используя локальную схему и локальное пространство имен (noNamespaceSchemaLocation).
3. Самостоятельно проверить валидность  xml-файла (например, введя данные не соответствующего типа).
4. Результаты работы показать преподавателю на уроке.
5. Ответить на предложенные вопросы.
